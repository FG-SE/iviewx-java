# Notes for the Eyetracking Library Tutorial #

## Prerequisites ##

 * Java 8

## Receiver based mechanisms ##

 * Find a common name to refer to ETReceiver, ETRecorder and ETStreamService
 * ETReceiver:      Infinite working loops, such as game loops.
 * ETRecorder:      Recording data for storage and post-analysis.
 * ETStreamService: Constructing complex computation networks, using multithreading and backpressure mechanisms.
                    Data can be sent back to a GUI thread by using shared-memory datastructures and timer based
                    updates, or reactive extensions for GUI frameworks, such as RXJavaFX.
