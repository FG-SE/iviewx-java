Create JNA binding as JAR with bundled library:

java -jar PATH_TO_JNAERATOR_JAR -mode Jar -runtime JNA PATH_TO_HEADER PATH_TO_LIBRARY

Useful options:
-noLibBundle : Do not bundle library files
-mode Directory : Creates binding files in a directory structure (no JAR)
